# 超时队列
按照时间因子对元素排序，支持自定义时间因子。

## 功能
1. 可选指定容量和动态调整容量，若容量为零则无数量限制。
2. 提供放入、取出、清空等方法。
3. 支持根据时间因子批量取出超时元素，以及取出所有元素。
4. 支持判断是否指定元素，以及弹出指定元素。
