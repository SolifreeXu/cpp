# 简介
超时队列支持自定义时间因子。

# 功能
1. 按照时间因子对元素排序，提供放入、取出、清空等方法。  
2. 支持根据索引移除元素，以及指定时间因子批量取出超时元素。

# 版本
* 语言标准：C++20
* 当前版本：v1.0.0

# 文件
* [TimeoutQueue.hpp](src/TimeoutQueue.hpp)：定义超时队列类模板TimeoutQueue。
* [test.cpp](test/test.cpp)：测试代码。
