# 简介
页面置换算法：最近最久未使用(Least Recently Used)。

# 功能
LRU队列按照最近访问顺序对元素排序，提供查找、放入、取出、清空等方法。

# 版本
* 语言标准：C++20
* 当前版本：v1.0.0

# 文件
* [LRUQueue.hpp](src/LRUQueue.hpp)：定义LRU队列类模板LRUQueue。
* [test.cpp](test/test.cpp)：测试代码。
