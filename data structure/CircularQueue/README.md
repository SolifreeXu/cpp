# CircularQueue
参照标准库，封装循环队列，可以补充标准库。

## 介绍
循环队列是容器，提供随机迭代器，兼容标准库功能和语言特性。例如以下功能和特性：
1. 迭代器iterator
2. 算法algorithm
3. 基于范围for循环

## 项目
主要目录结构如下所示：
* Source：源代码
* Sample：示例代码
* Binary：可执行程序

## 示例
源代码与示例代码可跨平台。  
示例代码位于Sample文件夹，支持于不同系统环境构建示例程序：
* Windows：使用Visual Studio 2022打开sln解决方案文件进行构建。  
* Linux：使用make直接构建示例程序。
